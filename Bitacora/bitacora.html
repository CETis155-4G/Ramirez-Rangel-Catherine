CSS:

En la clase de hoy vimos lo que es CSS, una parte importante sobre las paginas web diseñadas con HTML, para poder hacer uso de este si estos estan en diferentes carpetas se nececita hacer un vinculo que es el siguiete: src="style. css".

Con CSS nos ayuda a darle estilo a una página web, en donde se propagan en cascada; CSS significa: "Cascading Style Sheets", que tiene el siguiente significado: Cascading, que significa que los estilos que aplicamos a los elementos de una página web se propagan a los elementos que contiene.
